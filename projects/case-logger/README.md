# Procedure Case Logger

A simple tracking system for Dr. B's interventional and structural procedures.

## Purpose
- Track procedure volumes for CV/credentialing
- Monitor outcomes and complications
- Generate statistics on demand

## Categories
- **Structural:** TAVR, MitraClip, TEER, Watchman/LAAO, PFO/ASD closure
- **Coronary:** PCI, CTO, rotational atherectomy, IVUS/FFR
- **Peripheral:** Renal denervation, PAD interventions

## Data Fields
- Date
- Procedure type
- Access site (femoral, radial, etc.)
- Indication
- Outcome (success/partial/unsuccessful)
- Complications (none/minor/major)
- Notes

## Files
- `cases.csv` — Raw case data
- `stats.md` — Auto-generated statistics

## Usage
Add cases to cases.csv or use the web form (coming soon).
