# EKG Images Needed for STEMI Presentation

## Required Images

### STEMI Patterns
1. **Anterior STEMI** - V1-V4 elevation
2. **Inferior STEMI** - II, III, aVF elevation  
3. **Lateral STEMI** - I, aVL, V5-V6 elevation
4. **Posterior STEMI** - V1-V3 depression, tall R waves
5. **RV Infarction** - V4R elevation with inferior STEMI

### STEMI Mimics
6. **LVH with strain pattern**
7. **LBBB** (with and without STEMI using Sgarbossa)
8. **Early repolarization (BER)**
9. **Pericarditis** - diffuse STE, PR depression
10. **Brugada pattern** - Type 1
11. **Hyperkalemia progression** - peaked T → wide QRS → sine wave

### Diagrams
12. **Coronary artery anatomy** - LAD, LCx, RCA territories
13. **STEMI treatment algorithm flowchart**
14. **Timeline graphic** - symptom onset to cath lab

## Notes

- All EKG images should be high resolution (at least 1920x1080)
- Include rhythm strip if possible
- Label key features (ST elevation, reciprocal changes)
- For case studies, ensure patient info is de-identified
